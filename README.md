Hi, I'm Shah, a current junior at Queens College studying computer science. I'm interested in scalable web applications and learning about best web design practices. Currently learning and building with AI + cloud. 

### What I've built recently:

* [**Sante, AI-powered transcription and progress-note taking assistant for mental health professionals**]( https://github.com/shahjacobb/Sante)

Using *faster-whisper* and *LLama-2-70B*, this lets providers transcribe recorded sessions with patients and choose between different progress note/types for their records.
* [**Mango10k an AI-powered equity research tool** that uses a state of the art RAG pipeline](https://github.com/Chubbyman2/Mango10K)

Using *[Nomic embeddings](https://blog.nomic.ai/posts/nomic-embed-text-v1)* and [MongoDB Vector Search](https://www.mongodb.com/docs/atlas/getting-started/), this is a chatbot that ingests  regulatory filings (e.g. 10-Ks, and 10-Qs) and lets the user ask questions about the document and cut through the compliance noise
